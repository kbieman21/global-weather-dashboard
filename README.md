# global-weather-dashboard - Web Scraping Capstone

An interactive web dashboard that scrapes real-time weather data from around the world, cleans and stores it, and provides insightful visualizations.

# Project Overview
This capstone project demonstrates end-to-end data engineering and visualization skills by:
- Scraping live weather data using **Selenium**
- Cleaning and transforming data with **Pandas**
- Storing data in a **SQLite** database
- Building an interactive dashboard with **Streamlit**
